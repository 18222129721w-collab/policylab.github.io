project 1：住房政策工具数据库

住房政策工具数据库（Housing Policy Tool Database，简称HPTDB）由中国人民大学国家治理大数据和人工智能创新平台“政策工具数据库建设与政策评价”团队于2025年1月发起建设，定位为面向学术界的公益数据项目，旨在为住房政策研究、政策制定与效果评估提供基础数据支撑。目前数据库已开放使用申请。

一、数据范围与内容

HPTDB以1998年住房制度改革以来我国各地公开发布的住房政策为基础，围绕“政策工具—工具设定”两个层面，对1998—2024年地级市层级住房政策文本进行系统梳理与结构化整理。团队从近44.5万份相关政策文本出发，拆分形成约422万个文本块；在此基础上识别到105万个相关文本块，并进一步抽取形成40余万条、3万余种住房政策工具记录。 工具类型覆盖住房供给与保障、住房金融与税费政策、购房资格与交易限制（如限购、限售、限贷）、市场监管与价格管理等，并细化到以下关键设定：力度多大、对谁生效、在何种条件下生效。

如使用本数据库发表论文，请按以下格式引用：

张友浪, & 王培杰. (2026). 住房政策工具数据库 [Data set]. 项目组自发布。

Zhang, Y., & Wang, P. (2026). Housing Policy Tool Database [Data set]. Housing Policy Tool Database Project Team.

二、数据申请操作指南（版本：2026年1月30日）

为规范数据开放流程、保障申请者权益，请申请者在申请前仔细阅读并按要求提交材料。 申请流程：

  1. 填写申请表：《住房政策工具数据库数据使用申请表》需如实填写。
  2. 签字/盖章：申请者须本人手写签字/盖章确认，严禁代签；学生申请者需额外由导师签字。
  3. 扫描提交：将签字/盖章后的申请表完整扫描为 PDF，发送至项目官方审核邮箱：policymetricslab@163.com。
  4. 邮件标题格式：HPTDB数据申请-姓名-单位全称-部门/专业-身份（学生/教师/研究人员）例如：HPTDB数据申请-张三-XX大学-公共管理学院-行政管理专业-教师
  5. 审核反馈：数据提供方将在2周内反馈审核结果及数据获取方式，统一发送至申请者预留邮箱；未收到邮件即视为未通过，且不再单独告知。
  6. 其他说明：数据提供方保留对任何申请作出决定的权利；申请表下载与咨询方式以项目官网与官方渠道公布信息为准；本指南最终解释权归 HPTDB 项目组所有，若有调整将通过官网提前公示。

我们在数据库建设中投入了大量人力与算力，期待将其作为一份“学术公共品”，为推动政策工具理论深化与住房政策工具研究提供数据支撑。诚挚欢迎各领域学者使用并提出意见建议，以推动数据库迭代完善。
